- 采用 Next.js App Router 架构，`src/app/page.tsx` 为客户端交互核心（'use client'），负责状态管理与 UI 渲染。
- `src/app/api/generate/route.ts` 作为服务端 API 路由，处理文件上传至 Vercel Blob 并代理请求至火山引擎 Ark API。
- 样式层通过 `globals.css` 引入 Tailwind CSS v4，结合自定义 CSS 动画实现玻璃拟态视觉效果。
- 依赖方向：前端组件 -> 内部 API 路由 -> 外部存储服务 (Vercel Blob) -> 第三方 AI 模型服务。
